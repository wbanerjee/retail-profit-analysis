# Retail Profitability Optimization — Global Superstore Case

**End-to-end business analytics consulting project** using Excel, SQL, Power BI, and structured storytelling — modeled in an MBB-style format.

---

## Problem Statement

A large B2C retailer is experiencing inconsistent profitability across product categories and customer segments.  
This project analyzes 5,000+ orders from the **Global Superstore dataset** to:

- Identify profit drivers and margin leaks  
- Evaluate discount impact  
- Recommend pricing and regional strategies  

---

## Tools & Technologies Used

| Skill Area     | Tools Applied                           |
|----------------|------------------------------------------|
| Spreadsheet    | Excel (PnL modeling, scenario analysis) |
| Database       | SQL (MySQL Workbench, analytical queries)|
| BI Dashboard   | Power BI (interactive visuals, drilldowns) |
| Storytelling   | PowerPoint (client-style consulting deck) |

---


## Key Outcomes

-  Identified **Technology** as top profit driver ($145K+)  
-  Found that **orders with ≥30% discount** incur **net losses**  
-  Flagged **Furniture category** for low efficiency  
-  Recommended **Central and West regions** for focused expansion  
- Built 2 Power BI dashboards + MBB-style strategy deck
